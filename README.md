# 🍸 Bombs Bar

Website institucional de um bar de coquetéis tropicais.

## 🌴 Sobre o projeto

O **Bombs Bar** é um site desenvolvido com HTML e CSS que apresenta:

* Conceito do bar
* Eventos temáticos
* Cardápio de drinks
* Opções sem álcool
* Contato e reservas

O objetivo é criar uma experiência visual moderna e elegante, com foco em estética tropical e sofisticada.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts
* Font Awesome

## 🎨 Funcionalidades

* Layout responsivo
* Navegação suave (scroll)
* Design moderno com identidade visual própria
* Galeria de imagens
* Seção de eventos e menu
